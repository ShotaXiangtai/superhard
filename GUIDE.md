# SuperHard ガイド

このサーバーには **SuperHard** プラグインが導入されています。
バニラよりはるかに難しく、生き残るには戦略と経験が必要です。

---

## 脅威スコア（Threat Score）

あなたが強くなるほど、世界があなたに反応します。

| レベル | スコア | 色 | 世界の反応 |
|--------|--------|----|-----------|
| 穏やか | 0〜99 | 緑 | バニラと同程度 |
| 不穏 | 100〜299 | 黄 | モブHP+10%、跳躍・ストレイフ開始 |
| 敵対 | 300〜699 | 橙 | モブHP+25%、ブロック破壊・足場構築 |
| 激怒 | 700〜1499 | 赤 | モブHP+45%、包囲戦発生率上昇 |
| 天罰 | 1500〜 | 暗赤 | モブHP+70%、精鋭出現率最大 |

**スコアが上がる行動**
- モブを倒す（通常+2〜6、ボス+100〜200）
- 時間経過（1分ごと+3）

**スコアが下がる条件**
- 1ゲーム内日を生存する（−40）
- 死亡（現在スコアの30%減）

ログイン時にアクションバーで現在レベルが確認できます。

## 精鋭モブ（Elite Mobs）

通常モブの一部がランダムで精鋭化されます。名前に色が付いていたら要注意。

| 種類 | 名前の色 | HP倍率 | 速度 | 特徴 |
|------|---------|--------|------|------|
| **鍛錬済み** | 水色 | ×1.25 | +10% | 基礎強化のみ |
| **古の者** | 金色 | ×1.60 | +20% | 鉄装備付き、ダメージ+35% |
| **黙示録の** | 暗赤 | ×2.00 | +35% | ダイヤ装備、火耐性、耐性付与 |

精鋭を倒すと **鍛えの欠片** をドロップすることがあります（確率40%）。

脅威レベルが高いほど精鋭の出現率が上がります。

## モブの新AI

モブが賢くなっています。バニラと同じ戦い方は通用しません。

### ゾンビ系
- **跳躍攻撃** — 8ブロック以内で飛び込んでくる
- **ブロック破壊** — 壁や扉を破って追いかける
- **足場構築** — 段差を昇るため足元に石を積み上げる
- **仲間召喚** — 戦闘中に援軍を呼ぶ（脅威: 敵対以上）

### スケルトン系
- **ストレイフ（横移動）** — 射撃しながら左右に移動
- **武器切替** — 近距離に入ると弓から斧に持ち替える
- **バックステップ** — 接近されたら後退する

### クリーパー
- **透明接近** — 8ブロック以上離れているとき**透明になって近づいてくる**
- **炎の残留** — 爆発跡に炎が残る

### スパイダー
- **蜘蛛の巣設置** — 進路に蜘蛛の巣を張る（25秒後に消える）
- **スライム弾** — 遠距離からスローネス付きの弾を撃ってくる

### ウィッチ
- **モブバフオーラ** — 半径15ブロック内の全敵対モブに速度・耐性を付与
- **テレポート** — 近づくと逃げ、離れると近づいてくる

### エンダーマン
- **自動アグロ** — 見つめなくても16ブロック以内で敵対化
- **Vex召喚** — ダメージを受けるとVexを2〜3体呼ぶ
- **水の凍結** — 移動経路の水が氷に変わる

### パック戦術（全モブ共通）
同種のモブが攻撃されると、**半径20ブロック内の仲間も一斉にターゲットを向ける**。
囲まれたら逃げることを優先してください。

## 包囲戦（Siege Night）

ランダムな深夜（確率7%）に包囲戦が発生します。

```
⚔ 包囲戦 ⚔
モブの大群が迫ってくる！
```

このメッセージが出たら即座に防衛準備を。

**流れ**
1. 深夜0時 — 警告タイトル + ラッパの音
2. 3秒後 — **第1ウェーブ**: ゾンビ・スケルトンが拠点周辺にスポーン
3. 63秒後 — **第2ウェーブ**: クリーパー・ピリジャーが追加
4. 123秒後 — **第3ウェーブ** + **首領**（精鋭ゾンビ）が出現
5. 183秒後 — 残ったモブが消える、終了メッセージ

**拠点の判定**: 最後に寝たベッドの場所が拠点として扱われます。
複数プレイヤーが同じ拠点（50ブロック以内）にいる場合は1か所扱いです。

> 最低3ゲーム内日は間隔が空くので、毎晩は起きません。

## 鍛えの欠片（Tempered Shard）

精鋭モブを倒すと **鍛えの欠片**（紫の破片）をドロップします。

**使い方: アンビルで装備を強化**

```
アンビル左スロット: 強化したい装備（武器・防具・道具）
アンビル右スロット: 鍛えの欠片
→ 結果: 「鍛えし ○○」が完成
```

**「鍛えし」装備の効果**
- Unbreaking III が付与される
- 名前が金色の「鍛えし〜」になる
- 一度強化した装備には再使用不可

## よくある死に方

- **クリーパーが透明** → 近くで急に爆発する。音に注意
- **エンダーマンが先制攻撃** → 見ていなくても近づくと敵対化する
- **包囲戦の首領** → 精鋭ゾンビ（古の者）。1人で正面から戦わない
- **スケルトンの斧** → 近距離で弓から斧に持ち替える。シールドが有効
- **ゾンビの足場** → 柵や段差は安全ではない
