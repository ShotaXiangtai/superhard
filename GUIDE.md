# SuperHard ガイド

このサーバーには **SuperHard** プラグインが導入されている。
バニラより遥かに難しい。生き残るには戦略と仲間が必要だ。

---

## RAGE スコア

戦えば戦うほど、世界がお前に牙を剥く。

| レベル | スコア | 世界の変化 |
|--------|--------|-----------|
| **CALM** | 0〜99 | バニラとほぼ同じ |
| **AGITATED** | 100〜299 | モブ HP+10%、跳躍・横移動開始 |
| **HOSTILE** | 300〜699 | モブ HP+25%、ブロック破壊・足場構築 |
| **INFURIATED** | 700〜1499 | モブ HP+45%、レイド発生率上昇 |
| **WRATHFUL** | 1500〜 | モブ HP+70%、精鋭出現率最大 |

**スコアが上がる行動**
- モブを倒す (+2〜6)　ボス撃破 (+100〜200)
- 時間経過 (1分ごと +3)

**スコアが下がる条件**
- 1日生存 (−40)　死亡 (現スコアの30%減)

ログイン時に現在レベルが表示される。

## 精鋭モブ

通常モブの一部がランダムで強化される。名前の色が変わっているやつは要注意。

| 種類 | 表示 | HP倍率 | 速度 | 特徴 |
|------|------|--------|------|------|
| **シュラ** | 黄色 | ×1.25 | +10% | 跳躍と速度強化 |
| **ハーシャ** | 金色 | ×1.60 | +20% | 鉄装備付き、ダメージ+35% |
| **テンマ** | 紫 | ×2.00 | +35% | ネザライト装備、火耐性+耐性 |

出現時に**独特のサウンド**が流れる。聞こえたら戦闘準備をしろ。

倒すと **ソウルシャード** をドロップ（40%）。

---

## モブの行動

バニラと同じ戦い方は通用しない。

### ゾンビ系
- 跳躍攻撃（8ブロック以内で飛び込んでくる）
- ブロック破壊（壁も扉も壊して追いかける）
- 足場構築（段差を昇るため石を積み上げる）
- 仲間召喚（戦闘中に援軍を呼ぶ）

### スケルトン系
- ストレイフ（射撃しながら左右に移動）
- 近距離で弓から斧に持ち替える
- 接近されたらバックステップ

### クリーパー
- **透明になって近づく**（8ブロック以上離れると見えなくなる）
- 爆発後に炎が残る

### スパイダー
- 進路に蜘蛛の巣を張る（25秒で消える）
- スローネス付きのスライム弾を撃ってくる

### ウィッチ
- 半径15ブロック内の全敵対モブに速度・耐性バフをかける
- 近づくと逃げ、離れると近づいてくる

### エンダーマン
- **見つめなくても**16ブロック以内で敵対化
- ダメージを受けるとヴェックスを召喚
- 移動経路の水が氷になる

### パック戦術（全モブ共通）
攻撃されると半径20ブロック内の同種モブが一斉にターゲットを向ける。

## レイド

深夜、世界が牙を剥くことがある。突如として闇が動き、モブの群れが拠点を包囲する。

```
⚔ RAID ⚔
```

このタイトルとサウンドが出たら即座に防衛準備をしろ。

**流れ**
1. 深夜 — ウィザー・雷・ソニックブームの効果音とともに警告
2. 3秒後 — **Wave 1**: ゾンビ・スケルトンが拠点周辺に出現
3. 63秒後 — **Wave 2**: クリーパー・ピリジャーが追加
4. 123秒後 — **Wave 3** + レイドボス級の精鋭（ハーシャ）が出現
5. 183秒後 — 残存モブ消滅、終了

**拠点判定**: 最後に寝たベッドの位置。複数プレイヤーが同じ拠点（50ブロック以内）なら1か所扱い。

> 最低3日間は間隔が空く。毎晩は来ない。

## 天魔王（レイドボス）

> 「ひとりでは倒せない。仲間を集めろ。」

`/sh boss spawn` で運営がスポーンさせる特殊ボス。
将来的にはレア確率でレイド中にも出現予定。

**Phase I** (HP 100% → 65%)
- 全プレイヤーに雷撃（視覚エフェクト + ダメージ）
- 《シュラ》眷属を3〜4体召喚
- 稲妻の輪と轟音でスポーン演出

**Phase II** (HP 65% → 35%)
- **冥府の波動**: 全員にウィザー + 暗闇
- **業火の柱**: 2秒後に着弾する予告付き火柱
- 装備・速度がさらに強化

**Phase III** (HP 35% → 0%)**
- カオスエラプション: ランダムにファイヤーボールを連射
- **孤独への裁き**: 半径40ブロック内にプレイヤーが1人以下 → 毎秒HP2%回復
- 永続的な速度II・筋力II

**討伐報酬**
- 参加全員: ソウルシャード x12
- AuraSkills: Fighting +5000XP、Defense +2000XP

---

## ソウルシャード

精鋭モブを倒すと40%の確率でドロップする紫の欠片。

**使い方: アンビルで装備を強化**

```
アンビル左 : 強化したい装備（武器・防具・道具）
アンビル右 : ソウルシャード
→ [TEMPERED] 装備に変化
```

**[TEMPERED] 装備の効果**
- Unbreaking III が付与
- 金色の名前に変わる
- 一度強化した装備には再使用不可

## AuraSkills 連携

サーバーに AuraSkills が導入されている場合、自動で連携が有効になる。

- **スキルレベルに応じて**周囲のモブが強くなる
- 精鋭撃破でシュラ+150 / ハーシャ+400 / テンマ+900 XP
- 天魔王討伐でFighting +5000 / Defense +2000 XP

## 管理者コマンド

```
/sh status               サーバー状態・全プレイヤーのRAGEスコア
/sh threat [player]      指定プレイヤーのRAGEスコア確認
/sh setlevel <p> <0-9999> RAGEスコアを強制設定
/sh siege                レイドの状態確認
/sh elite [SHURA|HASHA|TENMA]  視線先のモブを精鋭化
/sh boss spawn           天魔王をスポーン（自分の位置）
/sh reload               コンフィグ再読み込み
```

## よくある死に方

- **クリーパーが透明** — 音に注意。近づいて急に爆発する
- **エンダーマンの先制** — 見ていなくても16ブロック以内で敵対化
- **テンマ精鋭** — 出現音がしたら逃げるか集まれ
- **天魔王ソロ挑戦** — Phase 3 で毎秒2%回復。ひとりで挑んでも削り合いに勝てない
- **スケルトンの斧** — 近距離で弓から斧に持ち替える
