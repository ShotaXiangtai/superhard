# SuperHard ガイド

このサーバーには **SuperHard** プラグインが導入されている。
バニラより遥かに難しい。生き残るには戦略と仲間が必要だ。

---

## RAGE レベル

戦えば戦うほど、世界がお前に牙を剥く。

| レベル | スコア | 世界の変化 |
|--------|--------|-----------|
| **Lv.1** | 0〜99 | バニラとほぼ同じ |
| **Lv.2** | 100〜299 | モブ HP+10%、跳躍・横移動開始 |
| **Lv.3** | 300〜699 | モブ HP+25%、ブロック破壊・足場構築 |
| **Lv.4** | 700〜1499 | モブ HP+45%、レイド発生率上昇 |
| **Lv.5** | 1500〜 | モブ HP+70%、精鋭出現率最大 |

**スコアが上がる行動**
- モブを倒す (+2〜6)　ボス撃破 (+100〜200)
- 時間経過 (1分ごと +3)

**スコアが下がる条件**
- 1日生存 (−40)　死亡 (現スコアの30%減)

ログイン時に現在レベルが表示される。

## 精鋭モブ

通常モブの一部がランダムで強化される。名前の色が変わっているやつは要注意。

| 種類 | 表示 | HP倍率 | 速度 | 特徴 |
|------|------|--------|------|------|
| **Lv.1** | 黄色 | ×1.25 | +10% | 跳躍と速度強化 |
| **Lv.2** | 金色 | ×1.60 | +20% | 鉄装備付き、ダメージ+35% |
| **Lv.3** | 紫 | ×2.00 | +35% | ネザライト装備、火耐性+耐性 |

出現時に**独特のサウンド**が流れる。聞こえたら戦闘準備をしろ。

倒すと **鋼** をドロップ（40%）。

---

## モブの行動

バニラと同じ戦い方は通用しない。

### ゾンビ系
- 跳躍攻撃（8ブロック以内で飛び込んでくる）
- ブロック破壊（壁も扉も壊して追いかける）
- 足場構築（段差を昇るため石を積み上げる）
- 仲間召喚（戦闘中に援軍を呼ぶ）

### スケルトン系
- ストレイフ（射撃しながら左右に移動）
- 近距離で弓から斧に持ち替える
- 接近されたらバックステップ

### クリーパー
- **透明になって近づく**（8ブロック以上離れると見えなくなる）
- 爆発後に炎が残る

### スパイダー
- 進路に蜘蛛の巣を張る（25秒で消える）
- スローネス付きのスライム弾を撃ってくる

### ウィッチ
- 半径15ブロック内の全敵対モブに速度・耐性バフをかける
- 近づくと逃げ、離れると近づいてくる

### エンダーマン
- **見つめなくても**16ブロック以内で敵対化
- ダメージを受けるとヴェックスを召喚
- 移動経路の水が氷になる

### パック戦術（全モブ共通）
攻撃されると半径20ブロック内の同種モブが一斉にターゲットを向ける。

## レイド

深夜、世界が牙を剥くことがある。突如として闇が動き、モブの群れが拠点を包囲する。

```
⚔ RAID ⚔
```

このタイトルとサウンドが出たら即座に防衛準備をしろ。

**流れ**
1. 深夜 — ウィザー・雷・ソニックブームの効果音とともに警告
2. 3秒後 — **Wave 1**: ゾンビ・スケルトンが拠点周辺に出現
3. 63秒後 — **Wave 2**: クリーパー・ピリジャーが追加
4. 123秒後 — **Wave 3** + 精鋭 Lv.2 の首領が出現
5. 183秒後 — 残存モブ消滅、終了

**拠点判定**: 最後に寝たベッドの位置。複数プレイヤーが同じ拠点（50ブロック以内）なら1か所扱い。

## レイドボス

> みんなで戦おう！。

前回討伐から **12〜24時間後** にランダムでスポーンする。最低1日1回は必ず出現。

- 降臨まで **30分前 / 10分前 / 5分前 / 1分前** に全体アナウンス
- スポーン場所は最高 RAGE スコアのプレイヤーの近く
- 近くにいるプレイヤーのみ **BossBar** で HP が見える（80ブロック以内）
- ログイン時に次の降臨カウントダウンが表示される

**Phase I** (HP 100% → 65%)
- 全プレイヤーに雷撃（エフェクト + ダメージ）
- 精鋭 Lv.1 の眷属を3〜4体召喚
- 稲妻の輪と轟音でスポーン演出

**Phase II** (HP 65% → 35%)
- **冥府の波動**: 全員にウィザー + 暗闇
- **業火の柱**: 2秒後に着弾する予告付き火柱
- 装備・速度がさらに強化

**Phase III** (HP 35% → 0%)**
- カオスエラプション: ランダムにファイヤーボールを連射
- **孤独への裁き**: 半径40ブロック内にプレイヤーが1人以下 → 毎秒HP2%回復
- 永続的な速度II・筋力II

**討伐報酬**
- 参加全員: 鋼 x12
- AuraSkills: Fighting +5000XP / Defense +2000XP

## 鋼（素材）

精鋭モブを倒すと40%の確率でドロップ。装備を強化する素材として使う。

### 強化手順

1. **アンビルを設置して開く**
2. 左スロットに **強化したい装備** を入れる
   - 対応: 剣・斧・弓・クロスボウ・防具（ヘルメット〜ブーツ）・ツルハシ等
3. 右スロットに **鋼** を入れる
4. 出力スロットに **[TEMPERED] 装備** が出るので取り出す

```
[ 装備 ] + [ 鋼 ] → [ TEMPERED 装備 ]
```

### 強化後の変化

| 項目 | 変化 |
|------|------|
| 名前 | 金色の `[TEMPERED] ○○` に変わる |
| 耐久 | Unbreaking III が自動付与 |
| 再強化 | 不可（1回だけ） |

## AuraSkills 連携

サーバーに AuraSkills が導入されている場合、自動で連携が有効になる。

- スキルレベルが高いほど周囲のモブが強くなる
- 精鋭 Lv.1+150 / Lv.2+400 / Lv.3+900 XP
- レイドボス討伐で Fighting +5000 / Defense +2000 XP

## 管理者コマンド

```
/sh status                  サーバー状態・全プレイヤーの RAGE スコア
/sh threat [player]         指定プレイヤーの RAGE スコア確認
/sh setlevel <p> <0-9999>   RAGE スコアを強制設定
/sh siege                   レイドの状態確認
/sh elite [SHURA|HASHA|TENMA]  視線先のモブを精鋭化
/sh boss spawn              レイドボスをスポーン（自分の位置）
/sh reload                  コンフィグ再読み込み
```

## よくある死に方

- **クリーパーが透明** — 音に注意。近づいて急に爆発する
- **エンダーマンの先制** — 見ていなくても16ブロック以内で敵対化
- **精鋭 Lv.3** — 出現音がしたら逃げるか集まれ
- **レイドボスのソロ挑戦** — Phase III で毎秒2%回復。削り合いに勝てない
- **スケルトンの斧** — 近距離で弓から斧に持ち替える
